# E-Commerce Product Page

A simple E-Commerce Product Page built using HTML, CSS, Bootstrap, and JavaScript.

## Features

- Product image display
- Product title and price
- Product description
- Add to Cart button
- Dynamic cart counter
- Responsive design using Bootstrap

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## How It Works

When the user clicks the **Add to Cart** button, the cart counter in the navigation bar increases dynamically.
